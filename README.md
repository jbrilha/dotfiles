# dotfiles

## Includes dotfiles for neovim, tmux and ranger

### Neovim
Using [lazy.nvim](https://github.com/folke/lazy.nvim) to setup plugins, each in their own .lua file inside the plugins folder.
I might forget to update this but currently loaded are:
[nvim-autopairs](https://github.com/windwp/nvim-autopairs)
[nvim-colorize](https://github.com/norcalli/nvim-colorize)
[catppuccin](https://github.com/catppuccin/nvim)
[onedark](https://github.com/navarasu/onedark)
[comment](https://github.com/numToStr/Comment.nvim)
[cmp](https://github.com/hrsh7th/vim-cmp)
