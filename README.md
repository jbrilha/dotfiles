# dotfiles

## Includes dotfiles for neovim, tmux and ranger

### Neovim
Using [lazy.nvim](https://github.com/folke/lazy.nvim) to setup plugins, each in their own .lua file inside the plugins folder.
I might forget to update this but currently loaded are:
1. [nvim-autopairs](https://github.com/windwp/nvim-autopairs)
2. [nvim-colorize](https://github.com/norcalli/nvim-colorize)
3. [catppuccin](https://github.com/catppuccin/nvim)
4. [onedark](https://github.com/navarasu/onedark)
5. [comment](https://github.com/numToStr/Comment.nvim)
